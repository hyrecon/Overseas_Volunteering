# 🌏 3D Printing & Arduino Education Program

> PNU 31st Overseas Volunteer Group — Ratchaburi, Thailand

## 📌 About

This program was part of Pusan National University's overseas volunteering initiative, where we taught 3D printing and Arduino programming to local students at Phothawatthanasenee School in Chonburi, Thailand.

- Preparation: Dec 26, 2022 – Jan 28, 2023
- Dispatch: Jan 29, 2023 – Feb 15, 2023

---

## 🖨️ 3D Printing

### Printing

Tools: `Ender-5 Pro 3D Printer` · `Creality Slicer 1.2.3` · `Padlet` · `Meshmixer`

Example models from [Thingiverse](https://www.thingiverse.com/)

### Modeling

Tools: `Tinkercad` · `Fusion 360`

Students designed their own 3D models using Tinkercad:
1. Octopus
2. BWO

### 3D Pen Maker

Hands-on session using 3D pens for freeform creation.

---

## ⚡ Arduino Coding

Tools: `Arduino UNO` · `Arduino IDE 2.0.3` · `Tinkercad`

Students built circuits using Tinkercad, then wired them on real hardware:
1. LED blink
2. RGB LED control
3. Pull-up / Pull-down button
4. Potentiometer (variable resistor)
5. Button + LED control
